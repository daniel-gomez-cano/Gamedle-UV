
# Gamedle-UV: Aplicación Móvil de Adivinanzas de Videojuegos (Android/Kotlin)

## 1. Descripción del Proyecto

Gamedle-UV es una aplicación móvil desarrollada para Android en Kotlin, cuyo objetivo es permitir a los usuarios adivinar videojuegos a partir de imágenes de portada que se revelan progresivamente. La aplicación cuenta con dos modos de juego principales: SOLOPLAYER (individual) y MULTIPLAYER (multijugador competitivo de dos usuarios).

## 3. Configuración del Proyecto

### Firebase
Para proteger las llaves de API y la configuración sensible, el archivo `app/google-services.json` está excluido del control de versiones.

Para configurar Firebase en tu entorno local:
1. Descarga tu propio archivo `google-services.json` desde la consola de Firebase.
2. Colócalo en la carpeta `app/`.
3. Si no tienes uno, puedes guiarte por el archivo `app/google-services.json.example` para ver la estructura necesaria.
