
# Gamedle-UV: Aplicación Móvil de Adivinanzas de Videojuegos (Android/Kotlin)

## 1. Descripción del Proyecto

Gamedle-UV es una aplicación móvil desarrollada para Android en Kotlin, cuyo objetivo es permitir a los usuarios adivinar videojuegos a partir de imágenes de portada que se revelan progresivamente. La aplicación cuenta con dos modos de juego principales: SOLOPLAYER (individual) y MULTIPLAYER (multijugador competitivo de dos usuarios).
